Vagrant::Properties
===================

Management multiple machines.

Installation
------------

Add this line to your application's Gemfile:

```ruby
gem 'vagrant-properties'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install vagrant-properties
